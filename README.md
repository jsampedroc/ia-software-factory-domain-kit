# 🏫 Sistema Integral de Gestión Escolar (Vanguardia Edition)

Bienvenido a la solución líder para la gestión automatizada de instituciones educativas. Este software ha sido diseñado bajo los estándares más estrictos de ingeniería (Arquitectura Hexagonal y DDD) para garantizar seguridad, escalabilidad y facilidad de uso.

## 🚀 Inicio Rápido (Llave en mano)

No necesita instalar Java, bases de datos o compiladores. Este proyecto está completamente "Dockerizado".

### Requisitos Previos
- Tener instalado [Docker Desktop](https://www.docker.com/products/docker-desktop/) (Windows/Mac) o Docker Engine (Linux).
